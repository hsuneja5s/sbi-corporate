# State Bank of India — Corporate Landing Page

A modern, highly polished corporate landing page for the **State Bank of India (SBI)**. Built with HTML, semantic layout best practices, and premium custom CSS styling.

## 🚀 Features
- **Legacy & Trust Badges**: Spotlights over 200 years of heritage and public sector trust.
- **Interactive Journey Selectors**: Custom interactive tabs for Students, Salaried professionals, Business Owners, Retirees, and NRIs.
- **Tailored Products**: Clear recommendation cards for accounts, credit cards, and loans.
- **YONO Digital Suite**: Dedicated showcase for SBI's flagship digital banking platform.
- **Responsive Navigation & Footer**: High-quality design featuring standard legal and trust compliance guidelines.

## 🛠️ Development & Deployment
- **Styling**: Vanilla CSS (`style.css`).
- **Structure**: Semantic HTML5 (`index.html`).
- **Deployments**: Configured for instant deployment on **Vercel**.
